# folio.

> European AI-powered portfolio tracker

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Framework: Vite (auto-detected)
4. Click Deploy — done ✅

## Environment Variables (for production)

None required yet — Anthropic API key will be added when we build the backend proxy.
